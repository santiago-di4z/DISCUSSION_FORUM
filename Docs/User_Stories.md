# USER STORIES
- As an **unregistered visitor**, I want to **create a user account** so I can participate in the forum.
- As a **registered user**, I want to **create a thread** (a discussion within a board).
- As a **registered user**, I want to **comment on an existing thread** to participate in the conversation.
- As a **user**, I want to **see a list of boards** so I can choose a topic that interests me.
- As a **user**, I want to **see the most active threads** within a board to find the most relevant discussions.
- As an **administrator** or **system**, I want to **delete threads with no activity for 7 days** to keep the site clean and relevant.
- As a **reader** of a thread, I want to **identify when the original creator of the thread comments** so I can better understand the context of the discussion.
