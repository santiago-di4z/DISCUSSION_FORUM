# USER STORIES
- As an **unregistered visitor**, I want to **create a user account** so I can participate in the forum. [Implemented in Version 0.2]
- As a **registered user**, I want to **create a thread** (a discussion within a board). [Implemented in Version 0.3]
- As a **registered user**, I want to **comment on an existing thread** to participate in the conversation. [Implemented in Version 0.3]
- As a **user**, I want to **see a list of boards** so I can choose a topic that interests me. [Implemented in Version 0.3]
- As a **user**, I want to **see the most active threads** within a board to find the most relevant discussions. [Implemented in Version 0.3]
- As an **administrator** or **system**, I want to **delete threads with no activity for 7 days** to keep the site clean and relevant.
- As a **registered user**, I expect that only other registered users can participate in the forum.
- As a **reader** of a thread, I want to **identify when the original creator of the thread comments** so I can better understand the context of the discussion. [Implemented in Version 0.3]
